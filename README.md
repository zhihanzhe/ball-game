# 球球跳跃 - 3D 弹球游戏

基于 Three.js 的 3D 弹球游戏，5个关卡，节拍同步音频，黑洞机制。

## 运行方式

### 方式一：双击启动（推荐，无需安装任何东西）

进入 `dist/` 文件夹，双击 `start.bat`，浏览器自动打开游戏。

### 方式二：开发者模式（可修改代码）

```bash
npm install        # 首次需要安装依赖
npm run dev        # 启动开发服务器，浏览器打开 localhost:5173
```

### 方式三：构建生产版本

```bash
npm run build      # 输出到 dist/ 文件夹
npm run preview    # 预览构建产物
```

## 游戏玩法

- **鼠标左右移动** 控制球在轨道上的位置
- 落在**不同颜色**的方块轨道上 → 碎裂死亡
- 移出**轨道边缘** → 掉落虚空
- 连续跳跃达到目标次数 → 黑洞出现 → 吸入黑洞即可通关
- **白色发光轨道** → 吃到后获得10秒加速

## 5大关卡

| 关卡 | 名称 | 轨道类型 | 通关条件 |
|------|------|----------|----------|
| 1 | 启程 | 长直轨道 | 跳跃 1 次 |
| 2 | 虫洞 | 长直 + 两块 | 跳跃 10 次 |
| 3 | 视界 | 全部类型 | 跳跃 20 次 |
| 4 | 卡冈图雅 | 全部类型 | 跳跃 200 次 |
| 5 | 奇点 | 全部类型 | 跳跃 2000 次 |

## 技术栈

- **Three.js 0.160** — 3D 渲染引擎
- **Vite 5** — 构建工具
- **Web Audio API** — 节拍检测与音效
- **纯 JavaScript** — 无框架
